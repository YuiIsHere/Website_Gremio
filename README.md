Website para o Grêmio.
O arquivo "auxa" é o auxiliar do programa, que eu utilizo/ei para verificar e concertar bugs visuais nas páginas que não ocorriam em outras. 
Muitas partes do código (como qualquer outro algoritmo que um programador faz), foi copiado e alterado de outros sites quando eu não sabia como fazer uma certa ação. O site mais usado foi o w3schools. 
Espero que os comentários colocados no código tenham ajudado à torná-lo mais legível. Já que esse é o meu primeiro site "público", qualquer sugestão de código que for é bem-vinda (e pode ser feita nos formulários ou na página de discussões do GitHub).
Boa leitura!
